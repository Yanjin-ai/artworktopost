# artwork → post

一个极简的本地小工具：把自己的画作快速做成风格统一的 Instagram 图 / 幻灯片视频。

- 上传作品（按原图比例呈现，不裁切）
- 极简画框 + 统一字体 + 个性签名 + 创作理念
- 多幅作品组成系列：合集大图 / mp4 幻灯片
- 自动文案 + 话题标签，一键复制
- 中 / EN / 日 多语言
- 图片与草稿本地保存（IndexedDB），刷新不丢

## 使用
直接打开 `index.html` 即可（建议 Chrome）。无需安装、无需后端。

## 在线版
- GitHub Pages: 仓库 Settings → Pages → 选 `main` 分支（需仓库为 Public）
- 或拖到 https://app.netlify.com/drop 一键上线
